# The Global Kitchen API

## Tech Stack
- Runtime: Node.js
- Framework: Express.js
- Database: MongoDB
- ODM: Mongoose
- Configuration: dotenv

## Features
- Create recipes
- Get all recipes
- Filter recipes by category
- Update recipes
- Delete recipes
- MongoDB validation
- Global error handling
- Layered architecture
